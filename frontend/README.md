# E-Pharmacy Admin — Frontend

Eczane yönetim panelinin React + TypeScript + Vite ile geliştirilmiş arayüzü.

## 🛠️ Teknolojiler

- React 19 + TypeScript
- Vite
- React Router DOM
- React Hook Form + Yup
- Axios
- CSS Modules

## 🚀 Kurulum

```bash
npm install
npm run dev
```

Uygulama `http://localhost:5173` adresinde çalışır. Backend'in (`http://localhost:3001`) ayağa kalkmış olması gerekir.

## 📜 Komutlar

| Komut | Açıklama |
|-------|----------|
| `npm run dev` | Geliştirme sunucusu |
| `npm run build` | Production build |
| `npm run preview` | Build önizleme |
| `npm run lint` | ESLint kontrolü |

## 🗂️ Sayfalar

- `/` — Login
- `/dashboard` — İstatistikler, son müşteriler, gelir/gider
- `/orders` — Siparişler
- `/products` — Ürünler (ekle/düzenle/sil)
- `/suppliers` — Tedarikçiler (ekle/düzenle)
- `/customers` — Müşteriler (sayfalamalı)
